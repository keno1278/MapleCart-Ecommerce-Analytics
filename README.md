# MapleCart E-Commerce Analytics

## Overview
MapleCart is a simulated e-commerce analytics project focused on analyzing customer behavior, traffic sources, product performance, and conversion funnels using SQL.

## Traffic Sources Analysis

**Objective**  
Identify top traffic acquisition sources prior to optimization efforts.

**Key Insights**
- Google Search (nonbrand) was the dominant traffic driver
- Brand and Bing traffic contributed minimal volume
- NULL referrers indicate direct traffic or inconsistent tagging

---

## Channel Portfolio Analysis

**Objective**  
Understand user engagement patterns by hour of day and weekday.

**Key Insights**
- Traffic peaks during weekday business hours
- Weekends show significantly lower engagement
- Timing optimization can improve campaign effectiveness

![Channel Portfolio SQL Query](images/queries/channel_portfolio_query.png)

---

## Product Analysis

**Objective**  
Evaluate the impact of cross-sell implementation on cart behavior and revenue.

**Key Insights**
- Cross-sell improved cart engagement
- Revenue per cart session remained stable
- Average order value increased slightly post-implementation

![Product Analysis SQL Query](images/queries/product_analysis_query.png)

---

## Website Performance Analysis

**Objective**  
Analyze the end-to-end website funnel to identify conversion drop-offs.

**Key Insights**
- Strong progression to product and cart pages
- Major drop-offs at shipping and billing stages
- Checkout UX optimization presents conversion opportunity

![Website Performance SQL Query](images/queries/website_performance_query.png)
